## Getting Started

Mutual Mentor is a secure CRM designed specifically for financial advisors. It helps you track client relationships, monitor your activity metrics, set and achieve goals, and manage your sales pipeline efficiently.

### System Requirements

- Modern web browser (Chrome, Firefox, Safari, Edge)
- Internet connection
- Valid email address for account registration

---

## Creating Your Account

### Registration

1. Navigate to the Mutual Mentor application
2. Click "Create Account" or "Register"
3. Enter your information:
   - **Full Name**: Your display name in the system
   - **Email Address**: Used for login (must be unique)
   - **Password**: Must meet security requirements (see below)

### Password Requirements

For your security, passwords must:
- Be at least 12 characters long
- Include at least one uppercase letter (A-Z)
- Include at least one lowercase letter (a-z)
- Include at least one number (0-9)
- Include at least one special character (!@#$%^&*)

**Example of a strong password**: `MySecure2024!Pass`

### First Login

1. Enter your email address
2. Enter your password
3. Click "Sign In"
4. You'll be directed to your dashboard

## Managing Clients

### Adding a New Client

1. Navigate to **Clients**
2. Click **Add Client**
3. Fill in client information:
   - **First Name** (required)
   - **Last Name** (required)
   - **Phone Number**
   - **Email Address**
   - **Referral Source**: How they found you
   - **Notes**: Any relevant information

4. Click **Save**

### Client Pipeline Stages

Each client progresses through your sales pipeline:

| Stage | Description |
|-------|-------------|
| 1. Initial Contact | First outreach made |
| 2. Fact Finder Scheduled | Discovery meeting scheduled |
| 3. Fact Finder Complete | Discovery meeting completed |
| 4. Solution Presented | Recommendations presented |
| 5. Application Submitted | Paperwork submitted |
| 6. Policy Issued | Policy/account active |

### Advancing Pipeline Steps

1. Open a client's profile
2. View current pipeline position
3. Click **Complete Step** to advance
4. Optionally add notes about the interaction

### Searching and Filtering Clients

Use the search bar to find clients by:
- Name (first or last)
- Email address
- Phone number

Filter options:
- **Pipeline Stage**: Show clients at specific stages
- **Referral Source**: Filter by how they found you
- **Date Range**: Clients added within a timeframe

### Editing Client Information

1. Click on a client's name to open their profile
2. Click **Edit**
3. Update any fields
4. Click **Save Changes**

### Deleting a Client

1. Open the client's profile
2. Click **Delete**
3. Confirm the deletion

**Note**: Deleted data cannot be recovered. Consider exporting before deleting.

## Tracking Metrics

Consistent activity tracking is key to success in financial services.

### Available Metrics

| Metric | Description |
|--------|-------------|
| Dials | Outbound phone calls made |
| Meetings Set | Appointments scheduled |
| Kept Meetings | Appointments that occurred |
| Closes | New clients acquired |
| Plans | Financial plans created |
| Lives | Life insurance policies |
| Referrals | Referrals received |
| Fact Finders | Discovery sessions completed |
| Meetings Ahead | Pre-scheduled appointments |
| Points | Custom point tracking |
| Premium | Premium volume |

### Logging Daily Activity

1. Navigate to **Metrics**
2. Select the date (defaults to today)
3. Enter values for each metric
4. Click **Save**

### Viewing Historical Metrics

- **Daily View**: Activity for a specific day
- **Weekly View**: Totals for the current/selected week
- **Monthly View**: Monthly aggregations
- **Quarterly View**: Fiscal quarter summaries
- **Yearly View**: Annual totals

## Setting Goals

Goals help you stay focused and measure success.

### Goal Periods

- **Weekly Goals**: Reset each week
- **Monthly Goals**: Reset each month
- **Quarterly Goals**: Aligned with fiscal quarters
- **Annual Goals**: Full year targets

### Creating Goals

1. Navigate to **Goals**
2. Select the period type (weekly, monthly, etc.)
3. Enter target values for each metric
4. Click **Save Goals**

### Goal Progress

Your dashboard shows real-time progress:
- Green: On track or exceeded
- Yellow: Slightly behind pace
- Red: Significantly behind

## Bonus Challenges

Bonus challenges help you push beyond regular goals with time-bound objectives.

### Creating a Bonus Challenge

1. Navigate to **Bonuses**
2. Click **Create Challenge**
3. Configure:
   - **Name**: Challenge title (e.g., "Q1 Dial Challenge")
   - **Description**: Details about the challenge
   - **Metric Type**: Which metric to track
   - **Target Value**: Goal to reach
   - **Start Date**: When tracking begins
   - **Deadline**: Challenge end date

4. Click **Create**

### Tracking Progress

- View current progress on the Bonuses page
- Progress percentage shows how close you are
- Days remaining countdown helps pace your efforts

### Challenge Statuses

| Status | Meaning |
|--------|---------|
| Active | Challenge in progress |
| Completed | Target reached |
| Archived | Manually archived or expired |

### Updating Progress

Manual progress updates:
1. Open the bonus challenge
2. Click **Update Progress**
3. Enter the current value
4. Click **Save**

**Note**: Some metrics auto-update based on your logged activities.

## Exporting Data

Export your data for backups, reporting, or analysis.

### Available Exports

| Export Type | Format | Contents |
|-------------|--------|----------|
| Clients | CSV | All client records |
| Metrics | CSV | Activity history |
| Full Export | JSON | Complete data backup |

### How to Export

1. Navigate to **Settings** or use the export menu
2. Select export type
3. Choose format (CSV or JSON)
4. Click **Download**

### Export Security

- Exports contain only YOUR data
- Exported files should be stored securely
- Contains PII - handle according to compliance requirements

## Settings & Customization

### Fiscal Year Settings

Configure when your fiscal quarters begin:
1. Go to **Settings**
2. Set Q1-Q4 start dates (MM-DD format)
3. Save changes

**Example**: If your fiscal year starts July 1:
- Q1 Start: 07-01
- Q2 Start: 10-01
- Q3 Start: 01-01
- Q4 Start: 04-01

### Week Start Day

Choose when your week begins:
- **Sunday**: Traditional week (Sun-Sat)
- **Monday**: Business week (Mon-Sun)

### Date Format

Choose your preferred format:
- MM/DD/YYYY (US format)
- DD/MM/YYYY (International)
- YYYY-MM-DD (ISO format)

### Timezone

Set your timezone for accurate date/time tracking.

## Security Best Practices

### Protecting Your Account

1. **Use a Strong Password**: Follow the requirements exactly
2. **Don't Share Credentials**: Your account is for you only
3. **Log Out When Done**: Especially on shared computers
4. **Report Suspicious Activity**: Contact support immediately

### Data Privacy

- Client data is encrypted and isolated
- Only YOU can see your clients
- Regular security audits ensure protection
- Compliant with GLBA, SOC 2, and FINRA requirements

### Session Security

- Sessions expire after inactivity
- Tokens refresh automatically during use
- Multiple failed logins trigger security measures

### Password Management

To change your password:
1. Go to **Settings**
2. Click **Change Password**
3. Enter current password
4. Enter new password (meeting requirements)
5. Confirm new password
6. Click **Update**

## Getting Help

### Common Issues

**Can't log in?**
- Verify email spelling
- Check password carefully (case-sensitive)
- Use password reset if needed

**Data not showing?**
- Refresh the page
- Check date filters
- Ensure you saved entries

**Export not downloading?**
- Check browser download settings
- Try a different browser
- Ensure popup blockers allow downloads

### Support

For technical issues or questions:
1. Document the problem (screenshots help)
2. Note any error messages
3. Contact your system administrator
